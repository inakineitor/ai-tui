# Move tsdown Commands to Config Files

## Summary

Consolidate tsdown CLI flags into config files for cleaner package.json scripts and better maintainability.

## Current State

| Package | Config File | CLI Flags | Issue |
|---------|-------------|-----------|-------|
| `@acme/core` | Yes (`tsdown.config.ts`) | Duplicated | Redundant - config and CLI both specify same options |
| `@acme/utils` | No | All in CLI | No config file exists |

## Changes

### 1. `packages/acme-core/package.json`

Simplify scripts since `tsdown.config.ts` already contains all build options:

**Before:**
```json
"build": "tsdown src/index.tsx --format esm,cjs --dts --external react",
"dev": "tsdown src/index.tsx --format esm,cjs --watch --dts --external react"
```

**After:**
```json
"build": "tsdown",
"dev": "tsdown --watch"
```

### 2. `packages/acme-utils/tsdown.config.ts` (create new)

```typescript
import { defineConfig } from "tsdown";

export default defineConfig({
  entry: {
    ".": "src/index.tsx",
  },
  format: ["cjs", "esm"],
  external: ["react"],
  dts: true,
  clean: false,
});
```

Note: No `banner` option (differs from core), no `target: false` (use default).

### 3. `packages/acme-utils/package.json`

**Before:**
```json
"build": "tsdown src/index.tsx --format esm,cjs --dts --external react",
"dev": "tsdown src/index.tsx --format esm,cjs --watch --dts --external react"
```

**After:**
```json
"build": "tsdown",
"dev": "tsdown --watch"
```

## Files to Modify

- `packages/acme-core/package.json` - simplify scripts
- `packages/acme-utils/package.json` - simplify scripts
- `packages/acme-utils/tsdown.config.ts` - create new
