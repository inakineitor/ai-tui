# Plan: Upgrade react-library.json to modern standards

## Summary
Update `packages/tsconfig/react-library.json` from ES2015/es6 to modern ESNext standards.

## Current State
```json
{
  "compilerOptions": {
    "jsx": "react-jsx",
    "lib": ["dom", "ES2015"],
    "module": "NodeNext",
    "target": "es6"
  }
}
```

**Issues:**
- `lib: ["dom", "ES2015"]` - from 2015, missing modern APIs like `.at()`, `.findLastIndex()`, `Intl.Segmenter`
- `target: "es6"` - also from 2015, misses modern syntax

## Proposed Update
```json
{
  "$schema": "https://json.schemastore.org/tsconfig",
  "extends": "./base.json",
  "compilerOptions": {
    "jsx": "react-jsx",
    "lib": ["dom", "dom.iterable", "ESNext"],
    "module": "NodeNext",
    "target": "ESNext"
  }
}
```

**Changes:**
- `lib`: `["dom", "ES2015"]` → `["dom", "dom.iterable", "ESNext"]`
- `target`: `"es6"` → `"ESNext"`
